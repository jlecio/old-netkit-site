Diario 2009

Costruzione di una macchina domU del GULZ per la gestione e la distribuzione degli IPv6. Riconfigurato l'indirizzo IPv6 del domU server web del GULZ. Messa in funzione di un domU per sostituire il server LDAP presente nella DMZ attualmente in funzione sul server "Serale". Fanno riferimento a tale servizio sia le piattaforme e-learning interne che quelle esterne come edave.it. La configurazione di questo server (Debian Lenny) prevede gia' il mail server e la gestione autofs su LDAP. Tutti i tentativi hanno dato esito positivo e ora e' alimentato dal server primario.
