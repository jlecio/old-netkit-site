World Wide Web technologies

For this module we'll use Netkit4TIC with the connettivity with the real network (read README file).

For ours purpose we'll use Wikipedia extensively. We follow chapter 35, 36 and 37 of Douglas Comer "Computer Networks and Internet".

(35) World Wide Web Pages and Browsing

  • The Network Architecture Client/Server is the base for the Browser, e.g. a User agent that retrives a Web page and interprets HyperText documents with Hypermedia.
  • URI is used to identify a resource and URL provides means of acting upon.
  • The language used is based on HTML element or a more recent XHTML.
  • The protocol is Hypertext Transfer Protocol where it uses MIME to encode/decode data. Inside the protocol there are hook for Web cache.
  • Usually different client are included in Browser as Plugin.
