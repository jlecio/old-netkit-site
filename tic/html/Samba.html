Modulo 9

Amministrazione e gestione di una rete

Samba

Un host con Samba installato può mascherarsi come server di una rete M$ offrendo i seguenti servizi:

  • condivisione di filesystem
  • condivisione di stampanti
  • dare supporto a Risorse di Rete
  • autenticazione dei client ad un dominio
  • risoluzione dei nomi WINS

La documentazione è disponibile presso la home page dove è possibile anche consultare il libro Using Samba tradotto in italiano dalla hopslibri che ne permette il download.
Le caratteristiche più importanti della release 3.0:

  • Supporto di Active Directory: Samba 3.0 è in grado di unirsi ad ADS come server membro e autentica utenti usando LDAP/Kerberos.
  • Miglioramento del supporto delle stampa in Windows 2000/XP/2003 incluso attributi di stampa presenti in active directory.
  • Supporto di caratteri UNICODE

Un test fatto da ITWEEK mostra che una macchina Samba3 server senza configurazioni particolari supera la stessa macchina con Windows 2003 server senza configurazioni particolari di un fattore 2.5. Leggi l'articolo e controlla i grafici.

Nel laboratorio virtuale abbiamo cercato di sperimentare alcune funzionalità della suite di Samba a partire da una seplice rete di due pc che usano la stessa configurazione mostrata nei materiali del MIUR per finire con due reti fisiche su cui è definito lo stesso workgroup.
