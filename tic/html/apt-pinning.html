Apt-Pinning

Nel caso in cui la versione attuale ti sembri troppo vecchia e tu voglia usare le versioni testing o unstable occorre modificare il file /etc/apt/sources.list come segue: 

# Stable
deb http://ftp.us.debian.org/debian stable main non-free contrib
deb http://non-us.debian.org/debian-non-US stable/non-US main contrib non-free

# Testing
deb http://ftp.us.debian.org/debian testing main non-free contrib
deb http://non-us.debian.org/debian-non-US testing/non-US main contrib non-free

# Unstable
deb http://ftp.us.debian.org/debian unstable main non-free contrib
deb http://non-us.debian.org/debian-non-US unstable/non-US main contrib non-free

Il passo successivo è quello di modificare il file /etc/apt/preferences come segue: 

Package: *
Pin: release a=stable
Pin-Priority: 700

Package: *
Pin: release a=testing
Pin-Priority: 650

Package: *
Pin: release a=unstable
Pin-Priority: 600

Spesso è necessario inserire nel file /etc/apt/apt.conf la seguente riga: 

APT::Cache-Limit "8388608";

Ora siamo configurati per installare software dalle varie distribuzioni. Per esempio per installare la zsh dalla testing basta dare il comando: 

# apt-get install zsh/unstable

e il gioco è fatto.

Remember

Ricordati di aggiungere in coda al file /etc/apt/sources.list il riferimento al sito Debian per i fix dei bachi di sicurezza: 

deb http://security.debian.org/ stable/updates main
