homework-01

Per il mese di giugno ho pensato ad un test/esercitazione di costruzione di una "retina". Il lavoro si divide in due progetti di cui solo il primo è "obbligatorio":

  • rete base: firewall auto-costruito, hub (o cavo incrociato) e server Linux.
  • rete MAN: IP pubblico dedicato esclusivamente a questo progetto, firewall auto-costruito completo di hard disk, hub/switch e server Linux. L'istituo Severi e Gramsci hanno la disponibilità per essere centri delle relative reti delle strutture a cui appartengono i relativi corsisti.

Per la realizzazione dello progetto base abbiamo a disposizione l'intero mese di Giugno e la classe virtuale, dove ci troveremo sicuramente. Le informazioni relative al firewall le trovi in bering con il puntatore agli eventuali drivers non presenti nel disketto Bering uClibc base. Per l'installazione del server Linux le informazioni le trovi nel modulo 7.
