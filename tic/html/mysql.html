Internet Server

MySQL

I due server della rete sperimentale sono configurati per essere uno la replica dell'altro. Delle slide che illustrano come configurare questa funzionalità le puoi trovare quì.

Il server slave è utilizzato durante le ore di laboratorio da due classi quinte che lo adoperano per costruire semplici schemi di database, programmi in Visual Basic e applicazioni Access. Per l'accesso al database da remoto esiste l'ottimo programma Open Source MySQLCC (MySQL Control Center) che ha le seguenti funzionalità:

  • costruzione interattiva di query all'interno di un editor
  • amministrazione dei database e delle tabelle
  • aministrazione del server stesso

Per l'uso con il linguaggio Visual Basic si possono usare componenti ADO (ActiveX Data Objects) collegati via ODBC attraverso MySQL ODBC (utilizzato anche da Access). I pc client dei laboratori sono macchine Windows 98 e 2000. Per un esauriente articolo che descrive una tipica applicazione solo su macchine M$ e con l'uso del database da parte di Word, Excel, VB e Access vai a VB project.
Per l'uso con il linguaggio java (J2SE) si sono fatti test con MySQL Connector/J il driver JDBC ufficiale per MySQL e anche con il classico ODBC Bridge. Sono stati testati anche applet che permettono l'accesso al database dall'interno di un browser.
