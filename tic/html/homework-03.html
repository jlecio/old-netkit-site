Lavoro per l'inverno

homework-03

Il test/esercitazione di questa volta sarà la costruzione di una rete in cui l'autenticazione avviene attraverso server ldap. Il lavoro consiste nella realizzazione di una mini rete privata consistente in un server e un client (estensibile a piacere con due server e un gruppo di client) in cui il servizio di autenticazione viene implementato con server LDAP.

Per la realizzazione dello progetto base da collocare temporalmente all'inizio dell'inverno abbiamo a disposizione la classe virtuale. Le informazioni relative alla configurazione di LDAP le trovi a partire dal modulo 9.
