PoPToP experience

We prepare a "delta" image: 

user@realHost$ qemu-img create -b XPPSP2.img -f qcow delta-XPPSP2.img

Now we fire the QEMU side: 

user@realHost$ ifname=`sudo tunctl -b -u knoppux`; \
               sudo modprobe kqemu; \
               export QEMU_SW="-usb -usbdevice tablet -kernel-kqemu"; \
               export QEMU_NET="-net nic -net tap,ifname=$ifname"; \
               qemu -m 140 -hda delta-XPPSP2.img $QEMU_SW $QEMU_NET
