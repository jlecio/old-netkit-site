Altri link utili:

  • The Internet Engineering Task Force (IETF)
  • port number assignment
    Nell'RFC 1700 sono contenuti i numeri di porta assegnati dalla Internet Assigned Numbers Authority (IANA)
  • RFCs La lista di tutti gli RFCs, incluso titolo e stato.
  • TCP/IP Headers
    Verifica con Ethereal la corrispondenza dei pacchetti nella tua rete con i seguenti Request For Comments RFC:
    • IP Datagram Header, (RFC 791), Internet Protocol
    • TCP Segment Header, (RFC 793), Transmission Control Protocol
    • ICMP, (RFC 792, Internet Control Message Protocol

    • Switched LAN, Baldi M. e Nicoletti P., Mc-Graw-Hill Italia Associates, Inc.
